# E_dashboard

E_dashboard is a full-stack web application built with a clear separation between backend and frontend.

## Project structure
- **back_end**: REST API built with Node.js (Express) for handling business logic and data.
- **front_end**: Modern frontend application for the dashboard UI.

## Features
- Clean backend/frontend separation
- Scalable project structure
- Easy to install and run locally

## Installation

### Backend
cd back_end  
npm install  
npm start  

### Frontend
cd front_end  
npm install  
npm start
